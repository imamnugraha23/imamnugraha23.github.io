Week 1 Friday Get to Know Adaptive Web Design (AWD) dan Responsive Web Design (RWB)



1. Apa itu adaptive dan responsive web design? Apakah kamu setuju bahwa teknik tersebut bagus? Mengapa atau mengapa tidak?

Responsive Web design adalah Design yang menyesuaikan dengan ukuran layar tidak peduli apa perangkat target. Responsif menggunakan media query CSS untuk mengubah gaya didasarkan pada perangkat target seperti jenis layar, lebar, tinggi dll, dan hanya salah satu penyesuaian untuk beradaptasi/dapat digunakan dengan layar yang berbeda. Sedangkan Adaptive web design, di sisi lain, menggunakan layout statis berdasarkan breakpoints yang tidak merespon setelah mereka awalnya dimuat. Adaptive bekerja untuk mendeteksi ukuran layar dan memuat tata letak yang tepat, misalkan kita menggunakan 5 perangkat dengan ukuran layar yang berbeda-beda maka kita harus menyesuaikan ukuran kelima perangkat tersebut terlebih dahulu. Teknik Tersebut bagus karena bisa menyesuaikan kebutuhan user, dengan media apapun yang mereka gunakan tampilan yang dihasilkannya akan tetap rapi

2. Apakah hadirnya smartphone dan tablet mengubah desain web secara signifikan? Bagaimana kira-kira teknologi baru akan mengubah desain web di masa depan?

Tidak terlalu mengubah design web secara signifikan, karena dengan adanya adaptive dan responsive web design tadi, perubahan tidak akan terlalu sulit dilakukan, teknologi baru mungkin akan mempermudah developer dalam mengubah desain web dan menyesuaikan device yang digunakan, sedangkan di sisi user mungkin akan lebih memudahkan mereka menggunakan desain web tersebut

3. Bagaimana kira-kira orang yang memiliki kekurangan fisik menggunakan website? Apakah kita perlu menyesuaikan website untuk mereka juga? Solusi apa yang dapat mengatasi keadaan tersebut?

Mereka mungkin akan kesulitan untuk menggunakan website, oleh karena itu kita sebagai developer harus bisa menyesuaikan kebutuhan user bagaimana pun keadaan mereka, kita analisis terlebih dahulu kekurangan fisik apa yang mereka miliki, misalkan user tidak bisa melihat maka, kita harus membuat website yang berdasarkan perintah suara, seperti siri dll.
