Selamat datang di website Saya

Perkenalkan nama Saya Imam Nugraha. Biasa dipanggil Imam.

Alasan saya ingin bergabung dengan Hacktiv8, Ingin menjadi web developer, karena selama ini saya bekerja di luar bidang IT/Developer, pengetahuan yang di dapat di lingkungan kerja saya / pekerjaan tersebut hanya berfokus pada kegiatan pekerjaan biasa saja, jenjang karir yang tidak jelas. Karena itu saya ingin memperdalam ilmu saya di bidang web developer dan memilih karir sebagai web developer. Tokoh idola Saya adalah Steve jobs karena menurut saya dia sangat inovatif. Kedepannya semoga kita bisa saling membantu di Hacktiv8 ini.

Saat kulah saya mengambil jurusan Teknik Informatika di salah satu Universitas Swasta di Bandung dan kemudian Lulus pada tahun 2014. Awal Tahun 2015 saya mulai bekerja di salah satu stasiun televisi Swasta di Jakarta, yang diawali dengan pelatihan militer di Secapa Bandung selama satu minggu, setelah bekerja selama kurang lebih satu tahun setengah saya memutuskan untuk resign dan bergabung bersama Hacktiv8
